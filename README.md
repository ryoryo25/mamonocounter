# 見たことない魔物カウンター

日向坂46が出演したライブの中から自分が参加したものを選択すると、参加したライブ数と参加した『見たことない魔物』の数を表示します。

# Contributing

1. 次のコマンドで環境構築します。
   ```shell
   git clone https://github.com/ryoryo25/mamonocounter.git
   cd mamonocounter
   npm install
   ```

2. 次のコマンドで`localhost`にサーバが立つので、適当に編集します。
   ```
   npm run dev
   ```

3. このリポジトリをforkしてプルリク投げてください。

**Issueも大歓迎です**
